# 🤖 AI Customer Support Agent

## 📌 Objective

This project is a simple **Agentic AI Customer Support Agent** developed using **Python, Streamlit, and Google Gemini AI**. It helps customers answer order and product-related questions using Tool Calling and AI.

---

## 🚀 Features

- 📦 Track Order Status
- 🛍️ View Product Details
- 🔍 Search Products
- 💰 Find Cheaper Alternatives
- 🤖 General AI Chat using Gemini AI
- 🛠️ Tool Calling
- ⚠️ Error Handling
- 📝 Logging Tool Calls
- 🎨 Modern Streamlit User Interface

## ▶️ Installation

### Clone the Repository

```bash
git clone https://github.com/surenderdhakar5/agentic-ai-assignment.git
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Create a `.env` File

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

### Run the Application

```bash
streamlit run streamlit_app.py
```
