# AI Customer Support Agent

## Objective

This project is a simple Agentic AI assignment that answers customer questions for an online store.

## Features

- Track Order Status
- Search Products
- View Product Details
- Find Cheaper Alternatives
- Error Handling
- Logging Tool Calls

## Project Structure

```
agentic-ai-assignment/
│
├── app.py
├── agent.py
├── tools.py
├── data.py
├── logger.py
├── requirements.txt
└── README.md
```

## How to Run

```bash
python app.py
```
